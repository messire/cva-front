# CV Frontend Service

Frontend-приложение для сервиса CV (Curriculum Vitae), построенное на React и Chakra UI.
Предназначено для визуализации профилей пользователей, управления профессиональным опытом и взаимодействия с CV Backend Service.

---

## 🛠 Технологический стек

- **Библиотека:** React 19
- **Сборка:** Vite 7
- **UI Framework:** Chakra UI 3
- **State Management:** Zustand
- **Routing:** React Router 7
- **Icons:** React Icons
- **HTTP Client:** Native Fetch (с кастомной оберткой apiFetch)
- **Линтинг:** ESLint

---

## 🚀 Быстрый запуск

Для локальной разработки и запуска проекта выполните следующие шаги:

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```
Приложение будет доступно по адресу: `http://localhost:5173`

### Сборка для Production
```bash
npm run build
```

---

## ⚙️ Конфигурация

Настройки осуществляются через переменные окружения в файле `.env`. 

Основные параметры:
- `VITE_PROXY_TARGET`: URL бэкенд-сервиса (по умолчанию `http://localhost:8080`). Используется для проксирования запросов `/api` во время разработки.

## 📈 Потенциальные улучшения (Backlog)

Проект находится в активной разработке. Ниже перечислены возможные направления дальнейшего улучшения и развития:

### 1. Улучшение пользовательского интерфейса
- **Редактирование профиля:** Полноценный интерфейс для создания и редактирования данных пользователя (сейчас в разработке).
- **Экспорт в PDF:** Возможность генерации и скачивания CV в формате PDF непосредственно из браузера.
- **Анимации:** Добавление плавных переходов и микро-взаимодействий с использованием Framer Motion.

### 2. Функциональность
- **Поиск и фильтрация:** Расширенные фильтры в каталоге профилей по навыкам и опыту.
- **Многоязычность (i18n):** Поддержка переключения языка интерфейса.
- **Темная/Светлая темы:** Дальнейшая полировка компонентов для идеального отображения в обоих режимах (Chakra UI 3 Color Mode).

### 3. Качество кода и Тестирование
- **Unit-тесты:** Покрытие компонентов тестами с использованием Vitest и React Testing Library.
- **E2E-тесты:** Интеграционные тесты на Playwright или Cypress.
- **TypeScript:** Миграция на TypeScript для повышения надежности и улучшения DX.

### 4. Наблюдаемость и Аналитика
- **Error Tracking:** Интеграция с Sentry для отслеживания ошибок на клиенте.
- **Performance Monitoring:** Мониторинг Core Web Vitals.
