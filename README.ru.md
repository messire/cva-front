# CV Frontend Service

Frontend-приложение для сервиса CV (Curriculum Vitae), построенное на React и Chakra UI.
Предназначено для просмотра и редактирования профилей, работы с публичным каталогом и взаимодействия с Backend-сервисом CV.

---

## 🛠 Технологический стек

- **Библиотека:** React 19
- **Сборка:** Vite 7
- **UI-фреймворк:** Chakra UI 3
- **State Management:** Zustand
- **Маршрутизация:** React Router 7
- **Иконки:** React Icons
- **HTTP-клиент:** Native Fetch (с обёрткой apiFetch)
- **Линтинг:** ESLint

---

## 🚀 Быстрый старт

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```
Приложение будет доступно по адресу: `http://localhost:5173`

### Сборка для production
```bash
npm run build
```

---

## ⚙️ Конфигурация

Настройки управляются через переменные окружения в файле `.env`.

Основные параметры:
- `VITE_API_BASE_URL`: URL backend-сервиса (по умолчанию `http://localhost:8080`). Используется для проксирования `/api` запросов в dev-режиме.

## 📈 Возможные улучшения (Backlog)

Проект находится в активном, портфолио-ориентированном состоянии.
